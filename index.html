⚙️ Setup Shaz AI Task Manager

Connect your Anthropic API key and Google Sheets to get started. This is a one-time setup — everything is saved locally.

Get yours at console.anthropic.com/keys. Stored locally in your browser only.

Leave blank to use offline mode. Follow steps below to set up Sheets:

1

Open your Google Sheet named "Shaz AI Task Manager" (create it if needed). Add columns: id, name, category, status, percent in row 1.

2

In the sheet, go to Extensions → Apps Script. Paste the code from the Comments section at the bottom of this HTML file.

3

Click Deploy → New deployment → Web App. Set "Who has access" to Anyone. Copy the web app URL and paste it above.
